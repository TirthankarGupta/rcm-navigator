# RCM Navigator

An intelligent Revenue Cycle Management (RCM) assistant designed to support payer-rule interpretation, documentation workflows, HCPCS-based checklist generation, and future document intelligence retrieval.

## Current Capabilities

- Secure multi-user login system
- Admin-controlled authentication
- Logout functionality
- HCPCS-based checklist generation
- Device and anatomy image mapping
- Payer rule visualization
- Persistent multi-format document ingestion architecture
- Support for:
  - PDF
  - CSV
  - TXT
  - XLS
  - XLSX
  - XLSM
  - DOCX

## Planned Enhancements

- Admin-only document upload controls
- Intelligent document retrieval
- Query-based knowledge extraction
- Semantic search
- AI-assisted payer rule interpretation
- Enterprise-scale document indexing
- Advanced RCM workflow intelligence
